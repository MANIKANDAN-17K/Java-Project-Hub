# ☕ Java Project Hub

Welcome to **Java-Project-Hub** 👨‍💻  

This repository contains Java projects built while learning Java from **basic to advanced** concepts.

The goal of this repository is to strengthen core programming skills, object-oriented design, and problem-solving ability using Java.

---

## 📚 What This Repository Covers

This repository includes projects based on:

### 🟢 Core Java Fundamentals
- Variables & Data Types
- Control Statements
- Methods
- Arrays
- Strings

### 🟡 Object-Oriented Programming (OOP)
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Interfaces
- Constructors

### 🔵 Java Collections Framework
- ArrayList
- LinkedList
- HashMap
- HashSet
- TreeMap
- Iterator

### 🟣 Advanced Concepts
- Exception Handling
- File Handling
- Multithreading
- Streams API
- Lambda Expressions
- JDBC Basics

## 🚀 How to Run

### 🔹 Using Command Line

1. Compile:
```
javac Main.java
```

2. Run:
```
java Main
```

### 🔹 Using IDE

- Open project in Eclipse / IntelliJ
- Run main class
